# 地球online成就殿堂 Windows v0.2.5

这是默认成就扩展版，只追加默认成就和同风格 SVG 徽章，不改 UI、不改存档结构、不影响既有本地档案。

## 新增默认成就

- 财富自由
- 风催雨
- 基建狂魔
- 环球食客
- 享苹果人生
- 空难
- 方向盘已绑定
- 午夜条例
- 小团体
- 纯爱战士
- 世界频道发言
- 我成 boss 了

## 说明

- 默认成就总数从 36 个增加到 48 个。
- 新增默认成就只会出现在首次打开的新档案中。
- 已经使用过旧版、已有本地档案的用户不会被自动插入新默认成就，避免影响现有成就墙。
- 用户仍然可以自由修改、删除、切换达成状态。

## 下载

下载安装包 `EarthOnlineAchievementPalace-Setup.exe`，双击安装。它会覆盖第二版程序文件，但不会删除你的本地成就档案。
