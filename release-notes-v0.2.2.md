# 地球online成就殿堂 v0.2.2

这是一个安装升级修复版，功能和 v0.2.1 保持一致。

## 修复

- 修复安装新版时可能提示“对路径 runtime\node.exe 的访问被拒绝”的问题。
- 安装器现在会在覆盖程序文件前自动关闭第二版自己的后台 Node 服务。
- 关闭范围只限第二版安装目录里的 `runtime\node.exe`，不会影响第一版或电脑上其他 Node 程序。
- 删除旧程序目录时增加短暂重试，避免 Windows 文件句柄刚释放时覆盖失败。

## 下载

下载安装包 `EarthOnlineAchievementPalace-Setup.exe`，双击安装。它会覆盖第二版程序文件，但不会删除你的本地成就档案。
