# 地球online成就殿堂 v0.2.0

第二版独立发布，和第一版“地球online成就中心”并存，不互通、不覆盖。

## 新增

- 内置 36 个默认成就，第一次打开自动出现在陈列馆里。
- 默认成就初始全部为未达成，可修改、删除、切换达成状态。
- 默认成就自带黑金游戏徽章风格缩略图，大图和缩略图使用同一张资源。
- 安装目录改为 `C:\Users\你的用户名\AppData\Local\Programs\EarthOnlineAchievementPalace`。
- 本地档案目录改为 `C:\Users\你的用户名\AppData\Local\EarthOnlineAchievementPalace\achievement-archive`。
- 本地端口段改为 `3317..3399`，避免和第一版冲突。

## 下载

下载安装包 `EarthOnlineAchievementPalace-Setup.exe`，双击安装。安装完成后桌面会出现“地球online成就殿堂”快捷方式。

由于安装包未做商业代码签名，Windows 可能提示 SmartScreen；点击“更多信息”再选择“仍要运行”即可。
